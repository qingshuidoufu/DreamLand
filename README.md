## 所用技术

1. 框架：Spring、SpringMVC、Mybatis
2. 数据库：MySQL
3. 缓存数据库：Redis
4. 富文本编辑器：Kindeditor
5. 前端框架：zui、bootstrap、Jquery
6. 异步更新：Ajax
