DreamField
======

简介
----

网页链接: http://www.dreamfield.top

这是一个用来分享我的工作技术文章、旅行拍照或者其他破文章的一个网站。

简而言之，你可以把这个网站当作一个个人博客。当然！这个博客不仅只有我可以发言，其他用户也可以进行注册、登录、发文章等行为。

功能
----

1. #### 打开冰箱

   * 伸出小手手
   * 握住冰箱门
   * 打开冰箱门

2. #### 把大象装进去

   + 双手握住大象
   + 把大象塞进去

3. #### 关上冰箱门

   - 趁大象不注意
   - 使劲关门
